# HomeFix Agent Handbook

This folder documents how the HomeFix assistant operates across design, coding, and collaboration contexts.

## Files
- `shared-rules.md` — universal rules the agent follows in any project.
- `design-laws.md` — Edith/Edithia design principles for UI and theming.
- `codex-guide.md` — coding standards, testing expectations, and CLI usage.
- `chatgpt-guide.md` — conversation guidelines for the conversational agent.
